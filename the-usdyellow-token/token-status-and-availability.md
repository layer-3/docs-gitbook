# Token Status & Availability

This page shows which $YELLOW features are currently available in Yellow.

### Why this page exists

$YELLOW may be referenced across rewards, staking, governance, and other platform features.

Not all token-related features are always live at the same time.

Use this page to check:

* what is enabled now
* what is planned
* what is not yet available

### Feature status model

Yellow should label token features as:

* **Live**: available in the app now
* **Planned**: announced but not yet available
* **Limited**: available only to selected users/regions/features
* **Paused**: temporarily disabled

### Typical $YELLOW feature areas

Depending on platform rollout, $YELLOW may be used for:

* rewards
* staking
* fee offsets or rebates
* governance
* ecosystem integrations

Only features shown as **Live** should be treated as active product functionality.

### Source of truth

The source of truth for token feature availability is:

1. the Yellow app UI (if visible there)
2. this documentation
3. official announcements (if linked here)
