# Markets

A market is where you trade one asset against another.

Example: `ETH-USDT`

Each Yellow market has its own trading rules and settings. These help determine how orders are placed, matched, and charged.

Yellow markets define trading rules such as:

* tick size (price step)
* step size (size step)
* minimum order size or value
* fees
