# Token Integration Event (TIE) & Delivery

This page explains how the Token Integration Event (TIE) relates to token delivery for eligible participants.

### What the TIE is

The Token Integration Event (TIE) is the event referenced in the Token Purchase Agreement (TPA) that triggers token delivery and the applicable vesting schedule under the terms of that agreement.

### Why it matters

For participants covered by a TPA:

* token delivery timing is tied to the TIE
* vesting and restricted periods follow the rules in the TPA
* the TPA terms control in case of any summary mismatch
