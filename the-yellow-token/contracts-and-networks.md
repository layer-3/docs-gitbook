# Contracts & Networks

Use this page to verify the official $YELLOW token contract details before depositing, withdrawing, or trading.

### Verify before sending

Always confirm:

* token symbol
* network/chain
* contract address
* decimals

Sending to the wrong network or using an unofficial token contract can result in loss of funds.

### Official contract list

Publish one row per supported network:

* Network
* Contract address
* Symbol
* Decimals
* Explorer link
* Status (live/testnet)

### Token logo assets (optional but recommended)

Provide official logo files for wallets, partners, and integrators:

* SVG
* PNG (light background)
* PNG (dark background)

Include checksum and version/date if possible.

### Fake token warning

Yellow will never ask users to trust a token contract address from:

* direct messages
* unofficial chat groups
* unverified social posts

Use only the addresses listed in this page and in the app.
